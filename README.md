# JS Training from AntiGravity

Welcome to the JavaScript Training repository! This project contains code examples and exercises for learning JavaScript and Playwright.

## Project Structure

The repository is organized into chapters:

- **1_Chapter_01**: Covers the basics of JavaScript.
  - `01_Basics.js`
  - `02_Verify setup.js`
- **2_Chapter_02**: (Coming soon)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/anil-automation/JSTrainingfromAntiGravity.git
   ```
2. Navigate to the project directory:
   ```bash
   cd JSTrainingfromAntiGravity
   ```

## Prerequisites

- Node.js installed on your machine.
- A code editor like VS Code.

## Usage

You can run the JavaScript files using Node.js:

```bash
node "1_Chapter_01/01_Basics.js"
```
